## HTML 문법
- 이미지 첨부할 경우
```HTML
<IMG SRC = "https://item.kakaocdn.net/do/fc93afd0e2ae15fb1f03a87e6b0458dcf43ad912ad8dd55b04db6a64cddaf76d"/>
```

- Style 지정 방법
  - Head 부분에서 설정 가능
 ```HTML
body{
  background-color : gray;
  margin : 2%;
  padding : 1.5%;
}
```
  -  class, ID 지정할 경우
```HTML
.name{
  text-align : center;
}
#date{
  font-size : 15px;
}
```
